# StraPlay Clone

Clone do straplay.com integrado com a casa de apostas VemnaBet (https://vemnabet.bet).

## Features
- Home com cards dos jogos (cassino ao vivo + slots)
- Cada jogo abre uma página com gerador de sinais em tempo real + iframe da VemnaBet
- Ao abrir o iframe, o usuário é forçado a fazer registro/login no modal da VemnaBet
- Sinais aleatórios específicos por tipo de jogo (Roleta, Aviator, Bac Bo, Slots, Mines)

## Run local
```bash
npm install
npm start
# http://localhost:3000
```

## Deploy
Deployado no Railway via Dockerfile.
